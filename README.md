# Vibe Hub 🌌

A curated collection of experimental web experiences, interactive modules, and digital vibes.

> **Note:** This entire repository is **100% vibe-coded with AI**. Every line of code, every architectural decision, and every visual aesthetic was manifested through a collaborative dialogue between a human and the Gemini CLI agent.

## 🚀 Live Environment
Visit the live portal at: **[vibe.nuty.works](https://vibe.nuty.works)**

## 🧪 Current Modules
- **[Vibe Hub](https://vibe.nuty.works/)**: The high-performance, vanilla HTML entry point.
- **[Calculus Integration](https://vibe.nuty.works/calculus-integration/)**: An elegant, TeX-powered practice module for mastering anti-differentiation.
- **[Resistor Color Code](https://vibe.nuty.works/resistor-color-code/)**: A visual engineering tool for identifying resistor values and tolerances.

## 🛠 Tech Stack
- **Monorepo Architecture**: Managed via `npm` workspaces.
- **Frontend**: React + TypeScript + Vite (Vibes) / Vanilla HTML (Hub).
- **Styling**: Tailwind CSS v4.
- **Rendering**: KaTeX (Mathematics) / Custom CSS (Engineering).
- **Deployment**: Automated via GitHub Actions to Cloudflare Pages.

## 🧘 Vibe Coding Philosophy
This project follows a strict "Research -> Strategy -> Execution" lifecycle. We prioritize:
1. **Speed**: Instant-load Hub and CDN-cached math engines.
2. **Aesthetics**: High-contrast dark mode, bold typography, and technical precision.
3. **Isolation**: Each vibe is a self-contained module, seamlessly orchestrated into a single unified domain.
